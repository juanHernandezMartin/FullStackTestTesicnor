# Guía de Uso: Prueba Técnica Fullstack Juan Hernández Martín

Este proyecto es una aplicación web que utiliza Flask como backend y un frontend basado en HTML, CSS y JavaScript.
La funcionalidad principal es mostrar películas de Harry Potter obtenidas de la API de OMDb, con la posibilidad de filtrar por título, año y puntuación.

---

## 1. Requisitos previos

Para ejecutar este proyecto, asegúrate de tener instalado:
- Python 3.x
- Un navegador web moderno

---

## 2. Instrucciones de instalación y ejecución

1. **Instalar dependencias**  
    Abre una terminal en la carpeta raíz del proyecto y ejecuta:  
        pip install flask requests

2. **Inicializar la base de datos**  
    Ejecuta el archivo "FilmFinder.py" para iniciar el servidor Flask:  

    Luego, accede a la siguiente URL en tu navegador o utiliza 'curl' para inicializar la base de datos: 
        http://127.0.0.1:5000/init_db

3. **Obtener películas de Harry Potter**  
    Accede a la siguiente URL para poblar la base de datos con información de la API de OMDb: 
        http://127.0.0.1:5000/fetch_movies

4. **Abrir la aplicación web**  
    En tu navegador, abre la URL principal: 
        http://127.0.0.1:5000/
